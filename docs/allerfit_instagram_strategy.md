# 알러핏 인스타그램 전략 정리

## 1. 인스타그램 운영 목적

알러핏 인스타그램은 단순 홍보 채널이 아니라, 알러핏 브랜드의 첫 번째 검증 채널이다.

초기 목표는 팔로워 수를 빠르게 늘리는 것이 아니라, 음식 알레르기가 있는 사람들이 실제로 저장하고 공유하고 댓글을 남기는 문제를 찾는 것이다.

인스타그램을 통해 다음을 검증한다.

- 어떤 알레르기 주제에 반응하는가
- 어떤 음식/제품/외식 상황에서 불편함이 큰가
- 어떤 정보가 저장되는가
- 어떤 콘텐츠가 공유되는가
- 어떤 질문이 댓글과 DM으로 들어오는가
- 앱이나 커머스로 연결할 만한 니즈가 있는가

---

## 2. 채널 포지셔닝

알러핏 인스타그램은 **알러지 인류를 위한 먹거리 생활 가이드**로 포지셔닝한다.

### 핵심 문장

> 못 먹는 음식 말고, 먹을 수 있는 방법을 알려드려요.

### 채널 설명

> 계란 알레르기부터 시작하는 알러지 인류의 라이프스타일 채널. 성분표, 외식, 레시피, 간식, 제품 선택을 더 쉽게 도와드립니다.

### 프로필 소개 예시

```text
알러핏 AllerFit
알러지 인류의 라이프스타일 브랜드
🥚 계란 알레르기부터 시작해요
🍞 노에그 레시피 · 성분표 · 외식 체크
📌 먹을 수 있는 방법을 저장해두세요
```

## 3. 초기 운영 전략

## 3.1 초기에는 계란 알레르기에 집중한다

처음부터 모든 알레르기를 다루면 콘텐츠 메시지가 흐려진다.

초기 3개월은 계란 알레르기, 특히 계란 흰자 알레르기와 관련된 식생활 문제에 집중한다.

주요 주제:

- 계란 없는 레시피
- 계란이 들어가기 쉬운 음식
- 난백, 난황, 전란분 등 성분표 용어
- 빵집/카페/외식 체크리스트
- 편의점/마트 제품 확인법
- 알레르기 있는 사람의 공감 스토리

## 3.2 얼굴 출연 없이 시작한다

장비가 없고, 퇴근 후 육아를 마친 밤 시간만 활용할 수 있는 상황에서는 얼굴 출연이나 직접 요리 영상부터 시작하지 않는다.

초기 콘텐츠는 다음 포맷으로 만든다.

- 카드뉴스
- 한 장 저장형 이미지
- 체크리스트
- 공감형 텍스트 이미지
- 제품 성분표 기반 정리 콘텐츠
- 댓글 질문 답변형 콘텐츠

## 3.3 주 3회 발행한다

초기 권장 발행량은 주 3개다.

매일 업로드를 목표로 하면 지속하기 어렵다. 주 3개를 3개월 유지하면 약 36개 콘텐츠가 쌓인다.

추천 업로드 요일:

| 요일 | 콘텐츠 유형 |
|---|---|
| 월요일 | 정보/성분표 콘텐츠 |
| 수요일 | 외식/체크리스트/공감 콘텐츠 |
| 금요일 | 레시피/제품/참여형 콘텐츠 |

## 4. 현실적인 제작 방식

## 4.1 일요일 밤 90분 제작 루틴

가장 현실적인 방식은 일요일 밤에 다음 주 콘텐츠 3개를 몰아서 만드는 것이다.

| 시간 | 작업 |
|---|---|
| 0~10분 | 다음 주 주제 3개 선택 |
| 10~30분 | ChatGPT로 원고 초안 제작 |
| 30~65분 | Canva/Figma 템플릿에 적용 |
| 65~80분 | 인스타그램 캡션 작성 |
| 80~90분 | 예약 업로드 또는 업로드 준비 |

## 4.2 평일 밤 최소 운영 루틴

일요일에 모두 만들지 못했을 경우에는 평일 밤에 쪼개서 진행한다.

| 요일 | 작업 | 예상 시간 |
|---|---|---|
| 월요일 밤 | 이번 주 주제 3개 정리 | 20분 |
| 화요일 밤 | 카드뉴스 1개 제작 | 40분 |
| 수요일 밤 | 한 장 콘텐츠 1개 제작 | 20분 |
| 목요일 밤 | 공감형 콘텐츠 1개 제작 | 20분 |
| 금요일 밤 | 캡션 정리/예약 업로드 | 20분 |

## 5. 필요한 도구

초기에는 장비를 사지 않는다.

필요한 도구는 다음 정도면 충분하다.

| 도구 | 용도 |
|---|---|
| ChatGPT | 원고 초안, 제목, 캡션, 해시태그, 릴스 대본 변환 |
| Canva 또는 Figma | 카드뉴스 디자인 템플릿 제작 |
| 인스타그램 앱 | 업로드, 댓글/DM 확인 |
| Notion 또는 Google Sheet | 콘텐츠 주제 관리 |
| 휴대폰 기본 카메라 | 제품 성분표 촬영, 간단한 자료 수집 |

## 6. 콘텐츠 템플릿

초기에는 디자인을 매번 새로 만들지 않는다. 템플릿을 만들어두고 내용만 바꿔야 지속 가능하다.

## 6.1 필요한 템플릿 5개

| 템플릿 | 용도 |
|---|---|
| 제목형 카드뉴스 | 정보 콘텐츠, 성분표 설명 |
| 체크리스트형 | 외식, 장보기, 주문 전 확인 |
| 비교표형 | 대체재, 제품 비교, 소스 비교 |
| 한 장 저장형 | 성분 단어, 질문 리스트, 요약 정보 |
| 공감 문장형 | 브랜드 스토리, 사용자 공감 콘텐츠 |

## 6.2 카드뉴스 기본 구성

```text
1장: 강한 제목/문제 제기
2장: 왜 확인해야 하는지
3장: 핵심 정보 1
4장: 핵심 정보 2
5장: 실생활 예시
6장: 주의사항
7장: 저장/댓글 CTA
```

## 6.3 한 장 저장형 구성

```text
상단: 제목
중앙: 리스트 5~8개
하단: 주의 문구 + 저장 유도
```

## 6.4 공감형 텍스트 구성

```text
상단: 공감 문장
중앙: 짧은 상황 묘사
하단: 알러핏의 관점 또는 질문
```

## 7. 콘텐츠 기둥

## 7.1 대체 레시피

계란 없이 만들 수 있는 음식과 대체재를 소개한다.

예시:

- 계란 없는 팬케이크
- 계란 없는 쿠키
- 계란 없는 머핀
- 계란 없는 프렌치토스트 스타일 레시피
- 계란 대신 쓸 수 있는 재료

## 7.2 숨은 계란 성분 정보

성분표에 숨어 있는 계란 관련 표현을 알려준다.

예시:

- 난백
- 난황
- 전란분
- 난백분
- 알부민
- 리소자임
- 마요네즈
- 머랭
- 커스터드

## 7.3 외식 체크리스트

외식 전 확인해야 할 질문과 상황별 체크포인트를 제공한다.

예시:

- 빵집에서 물어볼 질문
- 치킨 주문 전 확인할 것
- 피자 주문 전 확인할 것
- 튀김 먹을 때 확인할 것
- 소스/드레싱 확인법

## 7.4 제품/간식 큐레이션

편의점, 마트, 온라인에서 제품을 고를 때 확인해야 할 기준을 알려준다.

주의: 특정 제품을 소개할 경우 “안전하다”고 단정하지 않는다. 성분과 알레르기 표시는 변경될 수 있음을 항상 안내한다.

## 7.5 브랜드 스토리/커뮤니티

알러핏이 왜 시작되었는지, 어떤 문제를 해결하고 싶은지, 사람들이 어떤 음식을 다시 먹고 싶어 하는지 공유한다.

예시:

- 알러핏을 시작한 이유
- 빵 하나 고르는데 오래 걸리는 사람들
- 댓글로 먹고 싶은 음식 제보 받기
- 알레르기 때문에 포기했던 음식 이야기

# 1주차: 브랜드 소개와 문제 공감

## 콘텐츠 1. 알러핏을 시작한 이유

- 유형: 공감형 텍스트 이미지
- 목적: 브랜드 스토리 전달
- 핵심 메시지: 가까운 사람의 계란 흰자 알레르기 경험에서 시작한 브랜드
- 형식: 1장 또는 3장 카드뉴스

### 카드 구성

1. 알러핏을 시작한 이유
2. 빵 하나 고르는데도 성분표를 봐야 하는 사람들
3. 음식 알레르기가 있어도 먹는 즐거움을 포기하지 않도록

### 캡션 방향

```text
알러핏은 거창한 아이디어보다 가까운 사람의 불편함에서 시작했습니다.
계란 흰자 알레르기가 있으면 빵, 디저트, 외식 메뉴 하나도 쉽게 고르기 어렵습니다.
알러핏은 알러지 인류가 먹는 즐거움을 포기하지 않도록 돕는 브랜드가 되고 싶습니다.
```

### CTA

- 계란 알레르기 때문에 불편했던 음식이 있다면 댓글로 알려주세요.

## 콘텐츠 2. 계란 알레르기라면 성분표에서 이 단어 확인

- 유형: 정보 카드뉴스
- 목적: 저장 유도
- 핵심 메시지: 계란은 다양한 이름으로 성분표에 표시될 수 있음
- 형식: 6~7장 카드뉴스

### 카드 구성

1. 계란 알레르기라면 성분표에서 이 단어 확인
2. 난백: 계란 흰자
3. 난황: 계란 노른자
4. 전란분/난백분/난황분
5. 알부민/리소자임 등 계란 유래 성분
6. 마요네즈, 머랭, 커스터드도 확인
7. 구매 전 최신 성분표와 알레르기 표시 확인

### CTA

- 장볼 때 보려고 저장해두세요.

## 콘텐츠 3. 빵집에서 계란 알레르기라면 꼭 물어볼 질문 5가지

- 유형: 체크리스트
- 목적: 외식/구매 상황에서 바로 쓰는 콘텐츠
- 핵심 메시지: 빵은 반죽, 필링, 토핑, 윤기 코팅 등에 계란이 들어갈 수 있음
- 형식: 한 장 저장형 또는 5장 카드뉴스

### 질문 예시

1. 반죽에 계란이 들어가나요?
2. 크림이나 필링에 계란이 들어가나요?
3. 표면에 계란물을 바르나요?
4. 같은 도구나 오븐에서 계란 제품과 함께 만들었나요?
5. 알레르기 성분표를 확인할 수 있나요?

# 2주차: 계란 없는 레시피와 대체재

## 콘텐츠 4. 계란 들어간 줄 몰랐던 음식 7가지

- 유형: 정보 카드뉴스
- 목적: 숨은 계란 정보 전달
- 핵심 메시지: 계란은 빵과 케이크뿐 아니라 다양한 음식에 들어갈 수 있음
- 형식: 7장 카드뉴스

### 음식 예시

1. 마요네즈 소스
2. 돈가스/튀김옷
3. 일부 생면/파스타면
4. 커스터드 크림
5. 머랭 디저트
6. 일부 어묵/가공식품
7. 빵 표면 계란물 코팅

### 주의 문구

제품과 매장마다 다를 수 있으므로 반드시 성분표와 알레르기 표시를 확인해야 한다.

### CTA

- 의외였던 음식이 있다면 댓글로 알려주세요.

## 콘텐츠 5. 계란 대신 쓸 수 있는 재료 5가지

- 유형: 대체재 카드뉴스
- 목적: 노에그 레시피 관심 유도
- 핵심 메시지: 요리 목적에 따라 계란 대체재가 달라진다
- 형식: 6장 카드뉴스

### 대체재 예시

1. 바나나 으깬 것
2. 사과소스
3. 두부
4. 치아씨드/아마씨드 젤
5. 베이킹파우더+식초 조합

### 주의 문구

대체재는 음식 종류와 식감에 따라 결과가 달라질 수 있다.

### CTA

## 콘텐츠 6. 계란 없는 팬케이크 아이디어

- 유형: 레시피 카드뉴스
- 목적: “못 먹는 것”이 아니라 “먹을 수 있는 방법” 제안
- 핵심 메시지: 계란 없이도 팬케이크를 만들 수 있음
- 형식: 6장 카드뉴스

### 카드 구성

1. 계란 없는 팬케이크 가능할까?
2. 기본 재료: 밀가루, 우유 또는 대체유, 베이킹파우더, 오일
3. 계란 대신 사용할 수 있는 재료
4. 반죽 농도 맞추는 팁
5. 굽는 팁
6. 알레르기 성분은 재료별로 확인하기

### CTA

# 3주차: 외식과 제품 선택

## 콘텐츠 7. 외식할 때 이렇게 물어보세요

- 유형: 문장 템플릿
- 목적: 실제 대화에서 쓸 수 있는 표현 제공
- 핵심 메시지: 알레르기 문의는 구체적으로 해야 한다
- 형식: 한 장 저장형 또는 5장 카드뉴스

### 문장 예시

```text
계란 알레르기가 있어서 문의드립니다.
이 메뉴의 반죽, 소스, 토핑에 계란이나 계란 유래 성분이 들어가나요?
조리 과정에서 계란이 들어간 메뉴와 같은 도구를 사용하나요?
알레르기 성분표를 확인할 수 있을까요?
```
